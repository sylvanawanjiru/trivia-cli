# Trivia CLI Game

This is a JavaScript command-line trivia game created for my Software Engineering lab.

## Features
- Start and play a quiz game
- Timer for each question
- Overall countdown (total game time)
- Sequential questions
- Immediate feedback (correct/incorrect)
- Score summary at the end
- Uses arrays, functions, loops, timers, async/await

## How to Run the Game
1. Install Node.js
2. Open terminal inside the project folder
3. Run:

   node index.js

## Files Included
- index.js – main JavaScript game logic
- README.md – documentation
